MAX_WIDTH

TEXT --{
int setContainerSlot(position, slotId, itemName, count, metadata)
int setContainerSlot(x, y, z, slotId, itemName, count, metadata)

Sets the item in a slot.
Returns the number of items added or removed if there is already the same item in this slot or count if the slot is empty or an other item is in it or -65 if there is no valid container

- position: The position of the container
- x, y, z (int): The coordinates of the container
- slotId (int): The id of the slot
- itemName (string): The name of the item
- count (int): The items stack size
- metadata (int): The metadata of the item
}--
