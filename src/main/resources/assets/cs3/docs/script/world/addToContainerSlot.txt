TITLE addToContainerSlot
MAX_WIDTH

TEXT --{
int addToContainerSlot(position, slotId, count)
int addToContainerSlot(x, y, z, slotId, count)

Try to add items to a container. If the slot is empty, this does nothing.
Returns the number of items added or -1 if there was no valid container.

- position: The position of the container
- x, y, z (int): The coordinates of the container
- slotId (int): The id of the slot
- count (int): The number of items to add to the container
}--
