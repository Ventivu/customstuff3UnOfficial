TEXT --{
int removeFromContainer(position, itemName, count, metadata)
int removeFromContainer(x, y, z, itemName, count, metadata)

Try to remove items from a container.
Returns the number of items removed or -1 if there was no valid container.

- position: The position of the container
- x, y, z (int): The coordinates of the container
- itemName (string): The name of the item
- count (int): The number of items to remove from the container
- metadata (int): The metadata of the item
}--
