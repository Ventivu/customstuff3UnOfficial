TEXT --{
int addToContainer(position, itemName, count, metadata)
int addToContainer(x, y, z, itemName, count, metadata)

Try to add items to a container.
Returns the number of items added or -1 if there was no valid container.

- position: The position of the container
- x, y, z (int): The coordinates of the container
- itemName (string): The name of the item
- count (int): The number of items to add to the container
- metadata (int): The metadata of the item
}--
