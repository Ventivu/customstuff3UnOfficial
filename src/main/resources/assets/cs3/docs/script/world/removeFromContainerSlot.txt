TEXT --{
int removeFromContainerSlot(position, slotId, count)
int removeFromContainerSlot(x, y, z, slotId, count)

Removes items from a container slot. If the slot is empty, this does nothing.
Returns the number of items removed from the slot or -1 if there is no valid container

- position: The position of the container
- x, y, z (int): The coordinates of the container
- slotId (int): The id of the slot
- count (int): The number of items to remove
}--
